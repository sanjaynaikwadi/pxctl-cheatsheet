
## pxctl cheatsheet

**Cluster Status**

| Command | Description |
| --- | --- |
| pxctl cluster -h  | List supported commands |
| pxctl cluster list | List nodes in the cluster |
| pxctl cluster alerts | Show cluster wide alerts |

**Complete Status**

| Command | Description |
| --- | --- |
| pxctl status  | Gives a complete output of cluster/nodes/storagepool |
